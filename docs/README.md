# 文档说明

本目录下的文档应与根目录 [README](../README.md) 保持一致。

当前项目定位为企业级多模态知识库平台，主线能力包括：

- 知识库与成员协作
- 文件夹与文档管理
- 多模态内容摄取（文档已闭环，图片/音视频依赖外部 OCR/ASR 回流）
- RAG 问答
- 文档处理进度实时通知

以下内容不应再作为当前版本既有能力对外描述：

- 多租户
- 短信验证码登录
- Push Device / 设备推送
- 对外暴露的 Test API
- Agent 编排平台
- 混合检索与 rerank
- 视频关键帧理解
- 完整知识图谱

如需补充设计文档，建议围绕这几个主题展开：

1. 权限模型
2. 文档摄取链路
3. 音视频转写回流
4. 向量检索与聊天协议
5. 前后端接口契约
